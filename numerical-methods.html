Numerical Methods

Numerical Methods Library

Classical numerical algorithms as reusable modules.

Overview

A Python library implementing numerical algorithms from Numerical Analysis by Burden and Faires. Each algorithm is implemented as a reusable module. The library includes implementations of root-finding, interpolation, linear systems solvers, numerical differentiation and integration methods, and solvers for partial differential equations.

Modules

  • Root-finding — Methods for locating zeros of functions: bisection, Newton-Raphson, secant method, fixed-point iteration, and Müller's method.
  • Interpolation — Techniques for fitting curves through data points: Lagrange interpolation, Newton divided differences, Hermite interpolation, and cubic splines.
  • Linear Systems — Solvers for linear systems: Gaussian elimination, LU and Cholesky factorizations, and iterative methods like Jacobi and Gauss-Seidel.
  • Differentiation & Integration — Finite difference formulas for derivatives and Newton-Cotes methods for numerical integration.
  • Partial Differential Equations — Finite-difference solvers for PDEs: Poisson's equation, the heat equation via forward-difference, backward-difference, and Crank-Nicolson schemes, and a Black-Scholes options pricer.
